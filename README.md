# MagicMaidButHTML

> ⚠️ **이 프로젝트는 DOM 조작 / 타이머 학습 목적의 연습작입니다.**
> 
> ⚠️ **원본 MV의 모든 저작권은 원저작자에게 있으며, 문제 시 즉시 삭제합니다.**

> MIMI - Magic Maid 곡의 MV를 HTML에서 구현하기

👉 **[결과물](https://pyrite9.github.io/MagicMaidButHTML/)**



## 🇰🇷 한국어

### 소개

> 목적 : 동영상 재생 없이 HTML 만으로 MV를 재현할 수 있는가? 

### 사용 기술

- HTML/CSS/JS (순수, 프레임워크 없음)
- requestAnimationFrame 기반 타임라인 엔진
- CSS writing-mode (세로 쓰기)
- 프레임 시퀀스 스프라이트 애니메이션
- Google Fonts (Noto Serif KR)
- Web Audio (HTML5 Audio)

---

## 🇺🇸 English

### About

> Can we recreate a music video using only HTML, without a video player?

This is a fan-made learning project that recreates the MV of "Magic Maid" by MIMI feat. Kasane Teto SV using pure HTML/CSS/JS.

### Tech Stack

- HTML/CSS/JS (Vanilla, no frameworks)
- requestAnimationFrame-based timeline engine
- CSS writing-mode (vertical text)
- Frame sequence sprite animation
- Google Fonts (Noto Serif KR)
- Web Audio (HTML5 Audio)
